The Code Tool dashboard is the hub for the axiom.ai Code Tool. From here you can navigate to the Playground, view run reports, manage settings, and check your remaining runtime.

![The axiom.ai Code Tool dashboard](/docs/code-tool/getting-started/dashboard.png)

## Navigation
***

The left navigation has the following items:

- **Home.** The default landing tab when you sign in.
- **Run reports.** Lists your recent run reports. For details, see [run reports](/docs/code-tool/run-reports).
- **Playground.** Test and run scripts in the web app. See [Playground](/docs/code-tool/playground).
- **Settings.** Account and API key settings. See [generate an API key](/docs/code-tool/token-generation).
- **Color theme.** Toggle dark mode for the dashboard.

## Runtime
***

Your remaining runtime appears at the top-left of the dashboard, shown as `used minutes / total minutes`. Runtime resets at the start of each billing period; it doesn't carry over.
