The axiom.ai Chrome extension integrates with workflow tools like [Zapier](/docs/no-code-tool/integrations/zapier), [Make](/docs/no-code-tool/integrations/make), [n8n](/docs/no-code-tool/integrations/n8n), and [Relay.app](/docs/no-code-tool/integrations/relay-app); spreadsheets like [Google Sheets](/docs/no-code-tool/integrations/google-sheets) and [Excel](/docs/no-code-tool/integrations/excel); LLM providers like [ChatGPT](/docs/no-code-tool/integrations/chatgpt), [Claude](/docs/no-code-tool/integrations/claude), [Gemini](/docs/no-code-tool/integrations/gemini), [Sonar](/docs/no-code-tool/integrations/sonar), and [DeepSeek](/docs/no-code-tool/integrations/deepseek) (all via the [**Integrate AI**](/docs/no-code-tool/reference/steps/integrate-ai) step); and developer tools like [2Captcha](/docs/no-code-tool/integrations/2captcha) and [Puppeteer](/docs/no-code-tool/integrations/puppeteer). For anything else, build your own integration with the [**Trigger webhook**](/docs/no-code-tool/reference/steps/trigger-webhook) step. If you'd like an integration that isn't listed, [contact support](/customer-support).
