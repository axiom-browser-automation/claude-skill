Connect axiom.ai and Zapier in either direction. Trigger a Zap when an automation runs, or trigger an automation from any Zap. You can also pass data between the two.

## Trigger a Zap from an axiom.ai automation
***

Use the [**Trigger webhook**](/docs/no-code-tool/reference/steps/trigger-webhook) step or the [**Write javascript**](/docs/no-code-tool/reference/steps/write-javascript) step to fire a Zap when your automation runs. For step-by-step walkthroughs:

- [Trigger a Zap from axiom.ai with Trigger webhook](/guides/zapier#triggering-a-zap-using-an-axiom-ai-automation)
- [Trigger a Zap from axiom.ai with JavaScript](/guides/zapier#triggering-a-zap-using-an-axiom-ai-automation-javascript)

To send data with the trigger, see the [Sending data to Zapier examples](/guides/zapier#examples).

## Trigger an axiom.ai automation from Zapier
***

Use Zapier as a middleman between its library of apps and axiom.ai. When a Zap fires, it triggers your automation and passes data into it.

- [Trigger an axiom.ai automation from Zapier](/guides/zapier#triggering-an-axiom-ai-automation-from-zapier)
- [Send data from Zapier to axiom.ai](/guides/zapier#sending-data-from-zapier-to-axiom-ai)

## Trigger local runs with Zapier
***

The axiom.ai [API](/docs/developer-hub/api) always runs automations in the cloud. To run on the desktop instead, use the [trigger a desktop run with webhooks](/guides/trigger-desktop) workaround.
