Headless mode runs an automation in the background, without opening a visible browser window. Useful for desktop runs you want out of the way, or when running on a remote machine without a display.

> **Warning:** Some sites detect headless browsers and block them. If a site behaves differently in headless mode, run it with the visible browser instead.

![The headless mode settings panel in the axiom.ai Builder](/docs/settings/headless-axiom.ai.jpg)

## Enable headless mode
***

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Run headless** section.
3. Toggle `Run headless` on.
