## What to use the Remove HTML code step for
***

Use this step to strip all the HTML from a set of data. Useful when ['Scraping HTML'](/docs/no-code-tool/reference/steps/Get-data-from-website) and wish to extract data from the HTML.

You can use this step to:

- Remove HTML from scraped content
- Strip HTML from Google Sheet data

## How to configure the SRemove HTML code step
***

### Data

Click 'Insert Data' to select the data you wish to remove HTML from.
