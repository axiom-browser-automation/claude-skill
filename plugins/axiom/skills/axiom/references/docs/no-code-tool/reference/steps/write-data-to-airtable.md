## What to use the Write data to Airtable step for
***

Write data to Airtable from your Axiom steps to store, organise, and use your automation results in a structured table.

You can use this step to:

- Export [scraped](/docs/no-code-tool/reference/steps/get-data-from-website) data into Airtable
- Send [form submissions](/docs/no-code-tool/reference/steps/enter-text) directly to Airtable
- Store automation results in Airtable for later use

## How to configure the Write data to Airtable step
***

### Base ID

Input your airtable base ID.

### Table ID or name

Input the table ID or name.

### Data

Select the data to write.
