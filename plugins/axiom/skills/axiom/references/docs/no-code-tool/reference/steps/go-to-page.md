## What to use the Go to page step
***

The 'Go to page' step loads a web page for automating web actions or scraping data.

## You can use this step to:
***

- Load a page inside a nested [Loop](/docs/no-code-tool/reference/steps/loop) step
- Open a web page to scrape data

## How to configure Go to page
***

### Enter URL

Enter a URL or click 'Insert data' to pass URLs from a data source.
