axiom.ai stores each automation as a JSON file. Use these settings to download an automation as a file you can share or back up, or to import a previously downloaded file into the Builder.

::HeroMedia
::

## Import an automation
***

Importing replaces the current automation with the one in the file. Open the automation you want to overwrite first, or [create a new automation](/docs/dashboard#create-a-new-automation) and import into that instead.

> **Warning:** Importing overwrites the current automation as soon as you save. There's no undo, so make sure you're in the right automation before importing.

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Import or download** section.
3. Click **Select file** and choose the `.json` file to import.
4. Save the automation to apply the change.

![The import or download settings panel in the axiom.ai Builder](/docs/tutorials/share-automation.png)

## Download an automation
***

Downloading exports the current automation as a `.json` file you can share or back up.

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Import or download** section.
3. Click **Download**.

The file saves to your usual downloads folder. To use it elsewhere, follow the import steps above.
