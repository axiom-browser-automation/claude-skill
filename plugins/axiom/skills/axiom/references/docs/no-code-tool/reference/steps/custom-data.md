## What to use the Enter custom data step for
***

The Enter custom data step is a simple field for storing small amounts of data that can be made accessible to other steps via axiom.ai's 'Insert Data' tokens feature found inside step fields.

For example we use this step to store a list of CSS classes that we want to loop through in a [Click element](/docs/no-code-tool/reference/steps/click-element) step.

You can use this step to:

- Create a token with a number value
- Make a token with a text value
- Store data such as CSS classes

## How to configure the Enter custom data step
***

### Enter custom data

Enter a list of items to store as data; each line will be stored as a new row.
