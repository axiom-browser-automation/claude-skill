## What to use the Count rows step for
***

Use this step to count the number of rows in the selected data. Useful if you want to get the number of rows to control the number of loops you do with a Jump step. Use with the ['Read data from a Google Sheet'](/docs/no-code-tool/reference/steps/read-data-from-a-google-sheet-step) step.

You can use this step to:

- Control the number of times a Jump step jumps. Use this steps token in the 'Maximum cycles' field.
- Create a token for a row count to use in the ['Javascript step'](/docs/no-code-tool/reference/steps/write-javascript)
## How to configure the Count rows step
***

### Data

Select the data to count.
