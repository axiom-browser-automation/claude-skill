## What to use the End run step for
***

This step allows you to end your Axiom run without an error. You can use this in conjunction with conditional steps such as the [If step](/docs/no-code-tool/reference/steps/if-condtion) to end your automation when a certain condition is met.

You can use this step to:

- End run as part of a [If condition](./if-condtion)
- Stop a run with [Try catch](./try-catch)

The step requires no configuration.
