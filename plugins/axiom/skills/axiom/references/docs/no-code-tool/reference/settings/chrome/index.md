The Chrome settings control how axiom.ai launches the browser for an automation. Use them to [bypass bot detection](/docs/no-code-tool/reference/settings/chrome/bypass-bot-detection), use a [custom Chromium profile](/docs/no-code-tool/reference/settings/chrome/profile), point at a specific [Chrome executable](/docs/no-code-tool/reference/settings/chrome/executable-path), [load another extension](/docs/no-code-tool/reference/settings/chrome/load-extension), or run in [incognito mode](/docs/no-code-tool/reference/settings/chrome/incognito).
