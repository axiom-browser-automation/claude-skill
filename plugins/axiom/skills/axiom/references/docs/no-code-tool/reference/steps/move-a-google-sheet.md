## Introduction
***

The "Move a Google Sheet to a new folder in Google Drive" step can be used to move a Google Sheet within your Google Drive account. This can be used in combination with the [Create a new Google Sheet](./create-a-new-google-sheet) step to create sheets and then move them based on actions within your automation.

## Configuration
***

This step has two configuration options that are required.

### Sheet to move
***

Select the spreadsheet or paste in a URL of the sheet that you wish the step to move.

## Folder URL
***

Enter the URL of the destination folder for the spreadsheet. This must be created before the step is run and the folder must continue to exist for the duration of the automation runs.
