The maximum runtime option stops an automation if it has been running longer than the limit you set. This protects against runaway runs (loops that never exit, slow-loading pages, and so on) burning through your runtime allowance.

The default is set by your subscription plan.

> **Note:** This setting only affects cloud runs.

![The maximum runtime settings panel in the axiom.ai Builder](/docs/settings/runtime-axiom.ai.jpg)

## Set the maximum runtime
***

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Set maximum runtime** section.
3. Set `Maximum runtime` to the duration you want as the cap.
