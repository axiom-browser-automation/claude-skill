## What to use the Select random rows step for
***

Select random rows from a given data set, and create a new data set containing them. This could be used to vary content used in a direct message or to enter a random number.

You can use this step to:

- Enter random numbers stored in a Google Sheet randomly
- Enter text stored in a ['Google sheet'](/docs/no-code-tool/reference/steps/read-data-from-a-google-sheet-step) randomly

## How to configure the Select random rows step
***

### Rows

Enter the number of rows to select from.

### Data

Click 'Insert Data' to select the data to get rows from.

### Output

A preview of random results returned by this step.
