By default, axiom.ai ignores iframes (embedded frames within a page) when scraping or clicking. Iframes are usually unnecessary for the task and can introduce inconsistencies if the automation reaches into them by accident. Enable iframe support when you need to interact with content inside an iframe.

![The Interact with iframes settings panel in the axiom.ai Builder](/docs/settings/iframes--axiom.ai.jpg)

## Enable iframe support
***

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Interact with iframes** section under **Error handling**.
3. Toggle `Interact with iframes` on.
