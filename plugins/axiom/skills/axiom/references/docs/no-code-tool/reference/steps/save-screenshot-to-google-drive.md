## What to use the Save screenshot to Google Drive step for
***

Use this step to save a screenshot of a webpage to a Google Drive folder. This is a handy step when making Axiom bots to test websites or for data entry, and you want visual confirmation of the result. If you're going to run your bot on the desktop, try this [step.](/docs/no-code-tool/reference/steps/save-screenshot-locally)
You can use this step to:

- Capture the results of UI tests
- Confirm a web form has been submitted
- Grab a screenshot of data
- Help monitor status of a web page
- Test checkout pages

## How to configure the Save screenshot to Google Drive step
***

### Google Drive folder URL

Enter the URL of the folder in Google Drive to save the screenshot to.

### File name

Enter a name to save the screenshot as. Leave blank to use the default Google Drive filename. Only use 'Insert data' if you wish to pass a file name from another step like ['Read data from a Google Sheet'](/docs/no-code-tool/reference/steps/read-data-from-a-google-sheet-step).

### File type

File type to save the screenshot as a JPEG or PNG.
