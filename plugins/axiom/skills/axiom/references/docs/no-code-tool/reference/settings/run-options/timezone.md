By default, scheduled automations run in the cloud on UTC. Use the timezone option to run them on your local timezone instead.

![The timezone settings panel in the axiom.ai Builder](/docs/settings/time-zone-axiom.ai.jpg)

## Configure the timezone
***

1. Open the automation and click the **Cog** icon in the toolbar on the left.
2. Open the **Configure timezone** section.
3. Pick the timezone the schedule should use.

> **Note:** This setting only affects cloud runs. Desktop runs always use the local timezone of the computer they're running on.
