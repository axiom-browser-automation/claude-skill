## Scrape metadata
***

The "Scrape metadata" step can be used to extract metadata from a website. This metadata contains information about the page and is set by the author of the page. This includes social media tags, and general tags such as the page title, description or keywords. You can then use the data, or export this to an external service such as Google Sheets or Microsoft Excel.
