---
title: No-Code Tool
description: Build browser automations visually with the axiom.ai Chrome extension — no coding required.
order: 1
---

The No-Code Tool is a visual [Builder](/docs/no-code-tool/the-builder) inside the axiom.ai Chrome extension. Drag and drop steps to record clicks, scrape data, fill forms, and connect to Google Sheets — without writing code. Once your automation works, [schedule it to run](/docs/no-code-tool/how-it-works/schedule-and-run) on a recurring basis or trigger it from another app.
