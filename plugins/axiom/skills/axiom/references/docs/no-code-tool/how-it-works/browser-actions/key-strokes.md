To automate any keystroke or key combination, record it with the [**Press key(s)**](/docs/no-code-tool/reference/steps/press-keys) step.

::HeroMedia
::

## Use the Press key(s) step
***

Open the step finder, search for **press**, and add the **Press key(s)** step.

1. Click **Record**.
2. Press the key or combination you want to record.
3. Click **Stop recording**.

For worked examples, see the [keyboard shortcut guide](/guides/keyboard-short-cuts).

## Send keystrokes with JavaScript
***

The [**Write javascript**](/docs/tutorials/javascript) step can also simulate keystrokes. For security reasons, JavaScript-based keystrokes only work on the current web page.
