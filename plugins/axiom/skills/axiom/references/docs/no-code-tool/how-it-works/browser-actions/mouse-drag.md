To click and drag an element on a page, use the [**Mouse click & drag**](/docs/no-code-tool/reference/steps/mouse-click-and-drag) step. It emulates the action between two sets of coordinates.

::HeroMedia
::

## Use the Mouse click & drag step
***

Open the step finder, search for **drag**, and add the **Mouse click & drag** step.

1. Set the coordinates of the initial mouse click.
2. Set the coordinates to drag the mouse to.
