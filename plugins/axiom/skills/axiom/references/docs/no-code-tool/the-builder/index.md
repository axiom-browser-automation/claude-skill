The No-Code Tool is a visual [Builder](/docs/no-code-tool/the-builder) inside the axiom.ai Chrome extension. Drag and drop steps to record clicks, scrape data, fill forms, and connect to Google Sheets, no code required. This section covers everything you need to go from [installation](/docs/install) to [running your first automation](/docs/first-automation).

## Get set up
***

Install the Chrome extension and get familiar with the dashboard before you build your first automation. Every new account gets 120 minutes of free runtime so you can build and test without committing to a plan.

::DocsCards{path="install,dashboard" scope="file"}
::

## Use the Builder
***

The Builder is where automations come together. Add steps from the step finder, configure each one, and pass data between them with tokens. The selector tool helps you point steps at the exact elements on a page you want them to interact with.

::DocsCards{path="builder,pass, selector-tool" scope="file"}
::

## Share
***

Export an automation as a JSON file to share it with a teammate or back it up. Imported files replace the current automation, so import into a fresh automation if you don't want to overwrite an existing one.

::DocsCards{path="sharing" scope="file"}
::
