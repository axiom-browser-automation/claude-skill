::

## Dashboard overview
***

The dashboard is split into three areas, plus a vertical icon toolbar on the far left.

- **Icon toolbar (far left).** The burger menu opens [account](/docs/account) settings. Other icons cover home, run status, embedded docs, and video guides. Desktop and extension version numbers sit at the bottom.
- **Header (top).** Shows your runtime, subscription plan, and the **New automation** button.
- **Left column.** Menu links to recommended content, run reports, scheduled automations, Google Sheets, API keys, and your subscription plan.
- **Main column (right).** A searchable list of your automations.

![The axiom.ai dashboard with the icon toolbar, header, left menu, and automations list](/docs/tutorials/axiom-dasboard-vertical-icon-toolbar.jpg)

## Vertical icon toolbar
***

The icon toolbar gives you quick access to your account, the home dashboard, run status, and embedded help.

![The axiom.ai dashboard icon toolbar](/docs/tutorials/vertical-icon-toolbar-axiom-ai-dashboard.jpg)

### My account

From **My account** you can edit your email and password, manage your [subscription](/docs/subscription) plan, and delete your [account](/docs/account).

![The My account view in the axiom.ai dashboard](/docs/tutorials/my-account-axiom-ai.jpg)

1. Click the **burger** icon at the top left.
2. Click **My account**.

The home icon returns you to the default dashboard view, where your automations are listed.

The play icon shows the status of any currently running automations.

## Vertical menu
***

The vertical menu, to the right of the icon toolbar, changes the content shown in the main dashboard area.

### Dashboard

The default view, showing a list of your automations.

### Run reports

To check the status of all your runs, click **Run reports**. Reports load into the main dashboard area.

![Run reports view in the axiom.ai dashboard](/docs/tutorials/axiom-run-reports.jpg)

### Scheduled automations

To view your scheduled automations, click [Scheduled axioms](/docs/no-code-tool/how-it-works/schedule-and-run/automation).

![Scheduled automations view in the axiom.ai dashboard](/docs/tutorials/axiom-schedule-list.jpg)

### Google Sheets and API key

To connect or disconnect your [Google Sheets](/docs/no-code-tool/integrations/google-sheets) account, or fetch your [API key](/docs/developer-hub/api), click **Google Sheets and API key**.

![Google Sheets and API key view in the axiom.ai dashboard](/docs/tutorials/axiom-google-sheets-and-api.jpg)

### Change my plan

Out of runtime? Click **Change plan** to [upgrade](/docs/subscription) for more.

![The change subscription plan view in the axiom.ai dashboard](/docs/tutorials/axiom-manage-plan.jpg)

## Main column
***

The main column lists your automations and is shown by default. From here you can search, edit, run, and delete automations.

![The main automations list in the axiom.ai dashboard](/docs/tutorials/axiom-dasboard-vertical-icon-toolbar.jpg)

### Search for an automation

Use the search field to find an automation by name.

### Edit an automation

Click an automation in the list to open it in the [Builder](/docs/no-code-tool/the-builder/builder).

### Run an automation

Click the **Run** button next to any automation to start a run.

### Delete an automation

Click the **bin** icon to delete an automation.

> **Warning:** Deleted automations cannot be recovered.

## Dashboard header
***

The header sits across the top of the dashboard and contains the **New automation** button along with your runtime and subscription details.

### Create a new automation

The dashboard is the starting point for every automation. Click **New automation** in the top right corner to begin.

![The New automation button in the top right of the axiom.ai dashboard](/docs/tutorials/new-automation-dash.jpg)

### Runtime and subscription

Your [subscription plan](/docs/subscription) and runtime usage are shown in the top right corner. Runtime resets monthly and unused runtime does not carry over. Hover over the percentage to see a detailed breakdown.

![Hover the runtime percentage to see a detailed breakdown of runtime used](/docs/tutorials/axiom-dashboard-subscription-detials.jpg)
