The selector tool is how you tell axiom.ai which elements on a page to interact with or scrape. Use the [Single Selector](/docs/no-code-tool/the-builder/selector-tool/single) to target one element at a time, or the [Multi Selector](/docs/no-code-tool/the-builder/selector-tool/multi) to extract data from a list. Both support [custom CSS selectors](/docs/no-code-tool/the-builder/selector-tool/single#select-with-a-custom-selector) and can [accept selectors passed in from a data source](/docs/no-code-tool/the-builder/selector-tool/single#pass-selectors-from-a-google-sheet).

## Tutorials for the selector tool

Walk through each tool below. The Single Selector is the right starting point for most steps. Move on to the Multi Selector when you're working with lists or tables, and to custom CSS selectors when the visual selector picks an element that doesn't generalise across pages.
