The [**Write javascript**](/docs/no-code-tool/reference/steps/write-javascript) step lets you extend axiom.ai with custom JavaScript. The snippets in this section cover common patterns you can drop straight into a step, organised by category.

## Snippet categories
***

- [Data](/docs/developer-hub/snippets/javascript/data) for manipulating dates, scraped values, and CSV exports.
- [Interaction](/docs/developer-hub/snippets/javascript/interaction) for clicking, typing, and reaching into the Shadow DOM.
- [Navigation](/docs/developer-hub/snippets/javascript/navigation) for moving through browser history.
- [Network](/docs/developer-hub/snippets/javascript/network) for capturing network requests.
- [Scrape](/docs/developer-hub/snippets/javascript/scrape) for pulling data from a page.
- [Page manipulation](/docs/developer-hub/snippets/javascript/page-manipulation) for hiding or modifying page elements.
