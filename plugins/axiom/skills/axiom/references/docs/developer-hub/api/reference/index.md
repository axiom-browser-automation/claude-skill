The Reference section is the dev-lookup view of the axiom.ai API. Use it when you already know what you're looking for and just need the signature: a method and path on the [Endpoints](/docs/developer-hub/api/reference/endpoints) page, an HTTP status or error string on the [Error codes](/docs/developer-hub/api/reference/error-codes) page, or the schema of the input array on the [Data payload format](/docs/developer-hub/api/reference/data-payload-format) page.

For tutorials and worked examples, see the per-endpoint pages under [Run automations](/docs/developer-hub/api/run-automations) and [Browser sessions](/docs/developer-hub/api/step-functions).
